# Design System — Pokédex Search (Dark Theme)

## Color Palette

- **Background** (#0a0a0f): Deep navy-black dark mode base
- **Foreground** (#e8e8ef): Light silver text on dark backgrounds
- **Surface** (rgba(18,18,28,0.75)): Glassmorphism panel backgrounds
- **Surface Strong** (rgba(22,22,34,0.88)): Stronger glass panels
- **Border** (rgba(255,255,255,0.07)): Subtle glass borders
- **Accent Purple** (#a78bfa): Primary accent — buttons, badges, glows
- **Accent Blue** (#3b82f6): Secondary accent — info badges, links
- **Accent Red** (#ff3d3d): Damage badges, error states
- **Accent Yellow** (#facc15): Quick example tags, golden highlights
- **Success** (emerald-500): Search match badges, confirmations
- **Error** (red-500/20): Error borders, not-found states

## Typography

- **Body Font**: Inter (via next/font/google)
- **Heading Font**: Space Grotesk (via next/font/google)

## Effects

- **Aurora Background**: Animated radial gradients with slow breathing animation
- **Grid Overlay**: Subtle 60px grid lines fading downward
- **Glassmorphism**: backdrop-filter blur(24px) with translucent backgrounds
- **Shimmer Text**: Animated gradient text for hero headings
- **Floating Animation**: Gentle 4s float for Pokemon artwork
- **Pulse Glow**: Breathing glow on the Flee Rate stat card
- **Hover Lift**: translateY(-2px to -6px) on interactive cards
- **Gradient Buttons**: Purple-to-violet gradient with hover glow
